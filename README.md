# Vue 3 + Typescript + Vite 后台管理简易模板

## 使用到的相关技术：

- vue3
- vite
- typescript
- element-plus
- scss

## 开发

```bash
# 克隆项目
git clone https://github.com/Ritusan/vite-admin.git

# 安装依赖
npm i

# 启动服务
npm run dev
```

## 发布

```bash
# 发布
npm run build

# 预览
npm run preview
```
